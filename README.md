# キャリアポートフォリオ - Streamlitアプリケーション

内井優作のキャリアポートフォリオサイトです。Streamlitを使用して、スキル、職務経歴、プロジェクト実績などをインタラクティブに表示します。

## 特徴

- プロフィール情報の表示
- スキルセットの可視化
- 職務経歴のタイムライン表示
- プロジェクト実績の詳細表示
- 資格・学歴の一覧
- レスポンシブデザイン
- サイドバーナビゲーション

## 必要要件

- Python 3.8以上
- pip

## インストール

1. リポジトリをクローン:
```bash
git clone <repository-url>
cd yusaku-carrear
```

2. 依存パッケージをインストール:
```bash
pip install -r requirements.txt
```

## 実行方法

以下のコマンドでアプリケーションを起動:

```bash
streamlit run app.py
```

ブラウザで自動的に開かない場合は、`http://localhost:8501` にアクセスしてください。

## プロジェクト構造

```
yusaku-carrear/
├── app.py                 # メインアプリケーション
├── requirements.txt       # 依存パッケージ
├── README.md             # このファイル
├── claude.md             # プロジェクト説明
├── pages/                # 追加ページ(オプション)
├── assets/               # 画像やその他のアセット
└── data/                 # データファイル
```

## カスタマイズ方法

`app.py` を編集して、以下の情報を自分のものに変更してください:

1. **プロフィール情報** (20-22行目):
   - メールアドレス
   - GitHubリンク
   - LinkedInリンク

2. **自己紹介文** (35-47行目):
   - 自己紹介の内容
   - 専門分野

3. **スキル** (60-98行目):
   - プログラミング言語とレベル
   - クラウド & インフラのスキル
   - データベースのスキル
   - ツール & その他

4. **職務経歴** (104-134行目):
   - 会社名
   - 期間
   - 業務内容
   - 成果

5. **プロジェクト** (140-178行目):
   - プロジェクト名
   - 期間
   - 使用技術
   - 成果

6. **資格・学歴** (184-216行目):
   - 取得資格
   - 学歴

## 使用技術

- **Streamlit**: Webアプリケーションフレームワーク
- **Pandas**: データ処理
- **Plotly**: データ可視化(拡張用)
- **Pillow**: 画像処理
